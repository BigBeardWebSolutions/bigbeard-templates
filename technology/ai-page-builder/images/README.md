# Required Images for BBWS AI Page Builder Template

## Background Images

| Filename | Dimensions | Description |
|----------|------------|-------------|
| `hero-bg.jpg` | 1920x1080 min | Hero section background - dark, abstract, technology theme |
| `examples-bg.jpg` | 1920x1080 min | Examples section background - dark with subtle pattern |

## Logo Images

| Filename | Dimensions | Description |
|----------|------------|-------------|
| `logo.png` | 200x56 | Primary logo for header (transparent background) |
| `logo-white.png` | 200x57 | White version for dark footer background |

## Feature Icons

| Filename | Dimensions | Description |
|----------|------------|-------------|

## Image Guidelines

- All images should be optimized for web (compressed)
- Use PNG for logos and icons (transparency support)
- Use JPG for background images (smaller file size)
- Background images should work with dark overlay
- Icons should be consistent in style (outline or filled)
